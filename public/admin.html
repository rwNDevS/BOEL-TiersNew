Puntos por Peak Tier

Decide qué tier se utilizará para calcular los puntos del ranking.

Tier actual: si el jugador baja, también bajan sus puntos.

Peak Tier: se sigue mostrando el tier actual, pero los puntos se calculan con el mejor tier histórico obtenido en cada modalidad.

Cómo funciona el Peak Tier

El mejor tier queda almacenado aunque el jugador sea demoteado.

• El tier visible y editable siempre es el tier actual.

• El Peak Tier solo puede mejorar; nunca se reduce automáticamente.

• Al activar Peak Tier, el ranking general se recalcula de inmediato.

• Los tests normales, High Tests y cambios manuales actualizan el peak.

• Desactivar la opción no borra los peaks guardados; únicamente vuelve a calcular los puntos con el tier actual.
